📊 Durante minha pós-graduação em Data Science, na disciplina de Métodos Estatísticos, decidi ir além da teoria e aplicar os conceitos na prática.

Aproveitando os aprendizados das aulas, desenvolvi um projeto completo em Python com foco em Estatística aplicada à análise de dados, aprofundando meus estudos por meio de cursos complementares e construção de um notebook estruturado.

🔎 No projeto, trabalhei com:

Estatística descritiva (medidas de posição, dispersão e forma)
Padronização de dados com Z-Score
Probabilidade aplicada e Teorema de Bayes
Introdução à estatística inferencial
Visualizações e interpretação de dados voltadas à tomada de decisão

💡 O objetivo foi transformar conceitos estatísticos em aplicações reais, simulando desafios do dia a dia de um Analista de Dados.

Esse projeto reforça minha capacidade de:
✔ Aplicar teoria na prática
✔ Estruturar análises de ponta a ponta
✔ Identificar padrões e outliers com técnicas estatísticas
✔ Gerar insights a partir de dados
✔ Utilizar Python como ferramenta analítica
